## How to Use

1. Open `index.html` in any modern web browser
2. Navigate through sections using the top navigation menu
3. Click on any portfolio image to view it in the gallery modal
4. The hero carousel will automatically change images every 4 seconds

## Customization

### Replacing Images

To add your own fashion design images, replace the image URLs in `index.html`:

**Hero Carousel (lines ~30-50):**
```html
<img src="YOUR_IMAGE_URL_HERE" alt="Fashion Design">
```

**Portfolio Gallery (lines ~100-180):**
```html
<img src="YOUR_IMAGE_URL_HERE" alt="Design">
```

### Color Scheme

Modify colors in `styles.css` by changing the CSS variables at the top:
```css
:root {
    --primary-color: #d4af37;  /* Gold accent */
    --secondary-color: #1a1a1a; /* Dark background */
    --text-color: #333;
    --light-bg: #f8f8f8;
}
```
